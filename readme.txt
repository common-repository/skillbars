Easy animated Shortcode skill bars for WordPress.



== Description ==

Easy animated Shortcode skill bars for WordPress. A quick & easy way to add an responsive skill bars for WordPress. you can display unlimited skill bar in a same page via shortcodes.


View [DEMO](https://themepoints.com) for more details

<strong>Plugin Features </strong>

* Shortcode System.
* Unlimited Color.
* Fully responsive.
* All Browser support.
* Unlimited Skill Bars.
* Use via shortcode.
* Support post or page.
* And many more.

Here is the example Shortcode :
<code>
[skillbar title="HTML" percentage="50" color="#3498db" show_percent="true"]
[skillbar title="CSS" percentage="60" color="#2ecc71" show_percent="true"]
[skillbar title="PHP" percentage="80" color="#9b59b6" show_percent="true"]
[skillbar title="JQUERY" percentage="90" color="#e74c3c" show_percent="true"]
</code>

== Installation ==

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will use Shortcode to display skill bars post or page. use this shortcode <code>[skillbar title="HTML" percentage="50" color="#3498db" show_percent="true"]</code> to display skill bars any where post of pages.<br />
